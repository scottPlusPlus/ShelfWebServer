WebShelf

Open Source on GitHub

A quick HTTP webserver for stashing some data
Data is not persisted and will be lost when the server shuts down.
Make a PUT or POST request to /:table to append
Make a PUT or POST request to /:table/:itemKey to set

Example:

WebShelf.exe -port=3001
curl http://localhost:3001/foo -d "here is my data" //"success"
curl http://localhost:3001/foo //{"1":"here is my data"}
